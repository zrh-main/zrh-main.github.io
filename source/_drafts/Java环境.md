## Java虚拟机——JVM
* 概述:
  * JVM（Java Virtual Machine ）: Java虚拟机，简称JVM，是Java程序的运行环境，是Java 最具吸引力的特性之一。 
* 特性：
  * 跨平台: 软件的运行，都必须在操作系统之上，而Java编写的软件可以运行在任何操作系统上，这个特性称为Java**语言的跨平台特性**。该特性是由JVM实现的，我们编写的程序运行在JVM上，而JVM 运行在操作系统上。
